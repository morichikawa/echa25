# テストについて

## 自動テスト

```bash
npm test
```

## レイヤー機能の手動テスト

レイヤー機能はDOM操作が複雑なため、以下の手動テストを実施してください：

### テストケース

1. **レイヤー作成**
   - 「＋追加」ボタンをクリック
   - 新しいレイヤーが追加される
   - レイヤーリストに表示される

2. **レイヤー選択**
   - レイヤーをクリック
   - 選択されたレイヤーが青枠で表示される
   - 選択したレイヤーに描画できる

3. **レイヤー削除**
   - レイヤーの「×」ボタンをクリック
   - レイヤーが削除される
   - 最後の1つは削除できない

4. **複数レイヤー**
   - 3つのレイヤーを作成
   - それぞれに異なる色で描画
   - レイヤーが重なって表示される

5. **WebRTC同期**
   - 2つのブラウザで開く
   - 片方でレイヤーを追加
   - もう片方でも描画が同期される
