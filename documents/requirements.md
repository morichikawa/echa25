# プロジェクト要件

## プロジェクト概要

**echa25** - リアルタイム共同お絵描きアプリケーション

ブラウザ上で複数人が同じキャンバスに同時に絵を描くことができるWebアプリケーション。お絵描きチャットのような体験を提供する。

## 機能要件

### 必須機能
- ブラウザ上で動作するキャンバス
- 複数ユーザーによる同時描画
- リアルタイム同期（描画内容が即座に他のユーザーに反映される）

### 想定される追加機能
- ペンの色・太さの変更
- 消しゴム機能
- キャンバスのクリア
- ユーザー識別（色分けなど）

## 技術要件

### インフラ
- **サーバーレスアーキテクチャ**を採用
- AWS CDKによるインフラ管理
- 開発時のみリソースを起動、非開発時は完全削除可能な構成

### コスト最適化
- できる限り料金がかからない構成
- AWS無料利用枠を最大限活用
- 従量課金サービスの使用を最小限に

### タグ付け
- 全リソースに `echa25` タグを付与

## 運用要件

### デプロイ・削除
```bash
# 開発開始時
npm run deploy

# 開発終了時（全リソース削除）
npm run destroy
```

### 開発フロー
1. 開発開始時にCDKでリソースをデプロイ
2. 開発・テスト実施
3. 作業終了時にリソースを完全削除
4. コストは実際の使用時間のみ発生
